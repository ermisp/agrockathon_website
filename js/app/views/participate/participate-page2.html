Τί είναι hackathon?

Τα Hackathon είναι συγκεντρώσεις προγραμματιστών στις οποίες κατασκευάζονται λειτουργικές τεχνολογικές λύσεις σε πολύ περιορισμένο χρόνο.

Προέρχεται από το συνδυασμό των λέξεων hacking και marathon. Επίσης υπάρχει ως hack day, hackfest ή codefest.

Πρέπει κανείς να έχει υπόψιν ότι η λέξη hacking συχνά χρησιμοποιείται λάθος από τα MME. Στην πραγματικότητα μία από τις ερμηνείες που μπορεί να της δοθεί είναι η περιέργεια που είχαμε όλοι όταν ήμασταν παιδιά και ανοίγαμε τα παιχνίδια μας για να δούμε πως λειτουργούν.

Είναι μια διοργάνωση στην οποία συμμετέχουν προγραμματιστές ή άλλες ειδικότητες που σχετίζονται με την ανάπτυξη υλικού (hardware) ή λογισμικού (software) και συνεργάζονται εντατικά πάνω σε μία συγκεκριμένη ιδέα. Σε μερικές περιπτώσεις ένα hackathon μπορεί να έχει εκπαιδευτικό χαρακτήρα αλλά συνήθως έχει σκοπό την ανάπτυξη λύσεων που μπορεί μετέπειτα να υιοθετηθούν και να αναπτυχθούν πιο επίσημα. Η διάρκεια ενός hackathon μπορεί να είναι από μία ημέρα ως μια εβδομάδα στη διάρκεια της οποίας οι συμμετέχοντες έχουν την δυνατότητα να δουλεύουν ακατάπαυστα. Κατά συνέπεια ο ύπνος μπαίνει σε δεύτερη προτεραιότητα ενώ είναι σύνηθες τα γεύματα να γίνονται στα γρήγορα μπροστά στον υπολογιστή ή παρέα με άλλους διαγωνιζόμενους.

Ακολουθεί ένα απόσπασμα από τα συμπεράσματα του facebook για τα hackathon που έχει οργανώσει. “Τα Hackathons είναι τόσο πολύτιμα, επειδή είναι μια στιγμή που είμαστε σε θέση να σκεφτόμαστε πέρα από την επόμενη ημέρα ή εβδομάδα και απλά να χτίσουμε χωρίς περιορισμούς. Παίρνοντας αυτό το χρόνο για να σκεφτούμε διαφορετικά ανταποδίδει με νέες ιδέες και καινούργια πράγματα που υλοποιούνται Ακόμη και καθώς μεγαλώνουμε, ο αντίκτυπος των hackathons παραμένει μαζικός: το 60% των έργων από τα hacks που πραγματοποιήσαμε τον Δεκέμβριο του περασμένου έτους και τον Φεβρουάριο και τον Μάρτιο του τρέχοντος έτους έχουν ήδη χρησιμοποιηθεί, είτε εσωτερικά είτε σε χρήστες του Facebook.”

Γιατί να έρθω;

Για την αγροτική καινοτομία

  • Θα έχεις την ευκαιρία να πεις τη γνώμη σου για το πώς θα πρέπει να είναι τα επόμενα προϊόντα στον αγροτικό τομέα.
  • Θα μπορέσεις να παρουσιάσεις ποια είναι αυτά τα πράγματα στα οποία η αγροτική παραγωγή θα ήθελε τη βοήθεια της τεχνολογίας και με ποιο τρόπο.
  • Θα έχεις την ευκαιρία να αναπτύξεις ένα προϊόν που θα είναι χρήσιμο στους αγρότες και αποτελεί λύση σε κάποιο πραγματικό πρόβλημα.

Για την τεχνολογία

  • Θα γνωρίσεις τι μπορεί να κάνει η τεχνολογία όπως είναι σήμερα και τι είναι αυτό που έρχεται στο άμεσο μέλλον.
  • Θα έρθεις σε επαφή με προγραμματιστές που θα έχουν διαφορετικές ειδικότητες και εμπειρία

Για την επιχειρηματικότητα και την επαγγελματική δραστηριότητα

  • Για να δεις τις φάσεις μέσα από τις οποίες διαμορφώνεται μια ιδέα, αναπτύσσεται σε πρωτότυπο και τελικά σε προϊόν. Έτσι θα καταλάβεις κανείς ποιά είναι τα κόστη και η μελέτη για να γίνει κάποιο προϊόν. Τί είναι αυτό που κάνει ένα προϊόν ανταγωνιστικό (προσαρμογή στις ανάγκες κάποιου συγκεκριμένου επαγγελματία, υποστήριξη της γλώσσας, τεχνική υποστήριξη του προϊόντος)
  • Θα γνωρίσεις εταιρίες που να δραστηριοποιούνται ήδη στον αγροτικό τομέα και να τους μιλήσεις για το τί θα μπορούσε να βοηθήσει.
  • θα γνωρίσεις καινούργια πρόσωπα, επαγγελματίες και θα πάρεις μια ρεαλιστική εικόνα για το που πηγαίνει η αγορά εργασίας και συμβουλές για το που να ψάξεις τα επόμενα βήματα στην Ελλάδα ή στο εξωτερικό.

Για την εμπειρία

  • Για να γίνεις μέρος μιας ομάδας και να περάσεις καλά δουλεύοντας πάνω σε κάτι που σου αρέσει. Δεν υπάρχει τίποτα πιο ενδιαφέρον από το να δουλεύεις πάνω στις δικές σου ιδέες όχι γιατί πρέπει αλλά το θέλεις.
  • Για να ζήσεις το πως είναι ένα hackathon

Ποιά η διαφορά hackathon και agrockathon ?

Το Agrockathon είναι σαν ένα Hackathon αλλά εκτός από προγραμματιστές, θα συμμετάσχουν αγρότες, γεωπόνοι και τεχνολόγοι. Επίσης, το φαγητό θα είναι πολύ καλύτερο...!

Δανειζόμαστε την ιδέα ενός χάκαθον αλλά το αγκρόκαθον είναι κάτι περισσότερο.

Αυτό είναι το να αποφύγουμε να αναπτύξουμε ιδέες χωρίς να ξέρουμε αν θα ενδιαφέρουν κάποιον. Αλλά να βρεθούν στην ίδια ομάδα οι τελικοί χρήστες και να βοηθήσουν στην επιλογή των ιδεών που θα ήθελαν να δουν στο μέλλον να έρχονται στη διάθεση τους

Πρέπει να έχω φτιάξει ομάδα για να δηλώσω συμμετοχή;

Όχι, η συμμετοχή γίνεται ατομικά και οι ομάδες θα φτιαχτούν την πρώτη μέρα.

Όμως αν έχεις ήδη βρει άτομα με τα οποία σκοπεύεις να δουλέψεις δεν υπάρχει κανένα πρόβλημα να δηλώσεις συμμετοχή σαν ομάδα.

Σε αυτή την περίπτωση θα πρέπει να είστε ευέλικτοι και ανοιχτοί καθώς οι ομάδες θα είναι αποτέλεσμα των ιδεών και των ικανοτήτων που χρειάζονται σε κάθε πρότζεκτ.

Πρέπει να φέρω ότι υλικά μπορεί να χρειαστώ μαζί μου;

Θα προσπαθήσουμε να παρέχουμε τα βασικά υλικά που χρειάζονται για prototyping (arduino, raspberry pi, breadboards κ.τ.λ) όμως επειδή υπάρχει μεγάλη πιθανότητα να μην είναι αρκετά η συμβουλή μας είναι να προσπαθήσεις να έχεις μαζί σου ότι χρειάζεσαι

Αν έχω ήδη την ιδέα μου;

Αν η ιδέες μου είναι γύρω από software και δεν σκοπεύω να φτιάξω συσκευές;

Ο στόχος του Αγκρόκαθον είναι να αναπτυχθούν συσκευές ή συστήματα βασισμένα σε hardware (telematics, wireless sensor networks, internet of things) και low-level software (embedded linux, microcontrollers, embedded C). Κατα συνέπεια ψάχνουμε κόσμο που έχει εμπειρία στα embedded systems αλλά όχι μόνο.
