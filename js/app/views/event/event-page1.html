Οι στόχοι του αγκρόκαθον

Τα πρότζεκτ που θα αναπτυχθούν:
  • να σχετίζονται με την αγροτική καλλιέργεια με τα είδη καλλιέργειας που συναντώνται στην Κρήτη
  • να είναι γύρω από ότι έχει να κάνει με τα πρώτα στάδια της αγροτικής παραγωγής (καλλιέργεια, αποθήκευση προϊόντων)
  • σε δεύτερο βαθμό η μεταποίηση ή η προώθηση και οι αλυσίδες τροφοδότησης
